1. Paragraph tag --- Paragraph tags are used to display paragraph content.
  2. Heading tag ---- Heading tags are used to render the headings of the content.
  3. Title tag --- It sets the title of the webpage, which appears in the browser's title bar.
  4. Link tag ---- Specifies external resources like stylesheets and is commonly used to link to CSS files.
  5. a tag --- Creates hyperlinks to other web pages or resources.
  6. Img tag ---- Embeds images on the web page
  7. ul tag ---- Defines an unordered (dotted) list.
